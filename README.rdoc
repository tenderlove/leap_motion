= leap_motion

* http://github.com/tenderlove/leap_ruby

== DESCRIPTION:

Wrapper around the C++ api for Leap Motion.

== FEATURES/PROBLEMS:

* Meh

== SYNOPSIS:

  require 'leap_motion'

  controller = LeapMotion::Controller.new
  controller.each_frame do |frame|
    p frame
    puts "wow I got some data!~"
  end

== REQUIREMENTS:

* A Leap Motion

== INSTALL:

* gem install leap_motion

== HACKING:

1. Download a copy of the Leap SDK
2. Put the LeakSDK directory under ext
3. `bundle`
4. `rake compile`
5. `rake test`
6. Hack away
